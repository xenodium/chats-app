#+TITLE: Wasabi

[[file:./icon/wasabi_x200.png]]

* About

Wasabi is a native Emacs interface for WhatsApp messaging.

Wasabi communicates with a [[https://github.com/asternic/wuzapi][wuzapi]] process over standard I/O using json-rpc.

* Can I use it?

Not just yet. It's under development. Stay tunes.
